# 図解ラフ

スライドや動画で口頭説明するときの図解原稿。

---

## 図 1: MLOps パイプライン全体

```text
raw.properties
  -> Dataform
  -> feature_mart.property_features_daily
  -> embedding-job
  -> feature_mart.property_embeddings

search-api
  -> query embedding
  -> Meilisearch BM25 on Cloud Run
  -> BigQuery VECTOR_SEARCH
  -> RRF fusion
  -> rerank
  -> ranking_log

feedback_events + ranking_log
  -> training-job
  -> GCS model.txt
  -> training_runs
```

---

## 図 2: GCP 技術と工程の対応

```text
BigQuery: raw / feature_mart / embeddings / semantic retrieval / ranking_log / feedback_events
Dataform: feature engineering
Cloud Run Jobs: embedding-job / training-job
Cloud Run Service: meili-search
GCS: encoder / booster artifact
Cloud Run Service: search-api + in-memory TTL cache
Pub/Sub + Eventarc + Scheduler: retrain loop
```

## 図 3: 推論工程のシーケンス

```text
User

```
