## 伝え方の軸

- 「Phase 3 を作り直す」ではなく「**差分工事**」であることを一貫させる
- Vertex コンポーネントと Phase 3 コンポーネントを **横並び** で見せる (代替 vs 追加を区別)
- 残す判断 (VECTOR_SEARCH / Meili / Dataform) を「怠慢ではなく trade-off」として示す
- Port/Adapter 境界は **services/ranking.py が変わっていない** ことで示す
