# デモシナリオ

詳細版は `../03_デモシナリオ.md` を参照。ここでは撮影順だけを残します。

---

## 進行

1. `app/src/app/schemas/search.py` を開いて入力項目を確認
2. `app/src/app/services/ranking.py` を開いて主処理を確認
3. `app/src/app/adapters/candidate_retriever.py` を開く
4. `make ops-search` で実行
5. `make ops-ranking` で順位情報を確認
6. `mlops.training_runs` / `mlops.ranking_log` / `mlops.feedback_events` に触れる
7. `make ops-check-retrain` を実行

---

- `reasons`

## 一言メモ

- Cloud Run Service は `search-api`
- Cloud Run Jobs は `embedding-job`, `training-job`
- BigQuery は保存、特徴量、候補抽出、ログ保存を担当
- 再学習は Scheduler -> Pub/Sub -> Eventarc で起動
